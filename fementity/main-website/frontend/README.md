# Directory API

This is a generic CRA App with Chakra UI.

### Installing Dependencies

```
npm install
```

### Running Frontend

```
# Compile React App
npm run build
# Run the Webserver
npm start
```

## Dev Workflow

### Install Dev Dependencies

```
npm install
```
### Running the React App with Hotloading

```
npm run dev
```

### Formatting Code with Prettier

```
npm run prettier
```
